기본 편집 화면을 떠나지 않고 저장된 녹음과 가져온 파일을 정리하려면 `Workspace`를 사용하세요.

## 이런 경우에 적합합니다

- 저장된 항목을 별도 화면처럼 다루지 않고 한곳에서 다시 열고 싶을 때
- `All Items`, `Inbox`, 프로젝트 범위 사이에서 작업을 정리할 때
- 새 녹음이나 가져오기를 시작하기 전에 프로젝트별 기본값을 적용하고 싶을 때

## 먼저 세 가지 범위 이해하기

- `All Items`는 `Inbox`와 모든 프로젝트에 저장된 내용을 가로질러 보는 화면입니다.
- `Inbox`는 아직 프로젝트에 배정되지 않은 녹음과 가져오기 항목을 담습니다.
- 프로젝트는 전용 작업공간, 프로젝트별 기본값, 새 작업 진입점을 제공합니다.

## 프로젝트 만들기 또는 들어가기

1. `Workspace` 탭을 엽니다.
2. 왼쪽 rail에서 `All Items`, `Inbox`, 이름 있는 프로젝트를 전환합니다.
3. 클라이언트, 강의, 회의 시리즈처럼 반복되는 작업에 전용 공간이 필요하면 `New Project`를 클릭합니다.
4. 프로젝트 이름과 선택 설명을 입력한 뒤 만듭니다. 새 프로젝트는 custom icon 없이 시작합니다.
5. 새 항목이 자동으로 그 프로젝트에 남아야 한다면 `Live Record` 또는 `Batch Import`를 시작하기 전에 프로젝트를 엽니다.

## 저장된 항목 열기와 정리

1. 저장된 항목을 클릭하면 내장 detail pane에서 열리고 `Workspace`를 떠나지 않은 채 편집을 계속할 수 있습니다.
2. 검색, 필터, 정렬, `List View` / `Grid View` / `Table View`로 현재 범위를 좁힙니다.
3. 항목을 `Inbox`와 프로젝트 사이에서 이동하거나 여러 항목을 한 번에 삭제하려면 선택 모드를 켭니다.
4. 현재 프로젝트 context를 유지한 채 `Live Record` 또는 `Batch Import`를 시작하려면 프로젝트 헤더 버튼을 사용합니다.

## `Project Settings` 조정

1. 프로젝트를 열고 `Project Settings`를 클릭합니다.
2. 프로젝트 이름과 설명을 업데이트합니다.
3. 필요하면 프로젝트 아이콘을 고릅니다. 프로젝트 아이콘은 `New Project` modal이 아니라 여기에서 편집합니다.
4. 프로젝트 안에서 작업할 때 적용할 기본값을 고릅니다: `Default Summary Template`, `Default Translation Language`, `Default Polish Scenario`, 선택형 `Default Polish Context`, `Export Filename Prefix`.
5. 프로젝트별 인식 또는 정리 동작이 필요하면 `Text Replacement`와 `Hotword` set를 켜거나 끕니다.
6. 변경을 유지하려면 `Save`를 클릭하고, 프로젝트를 제거하고 항목을 `Inbox`로 돌리고 싶으면 `Delete Project`를 사용합니다.

## 참고

- `All Items`는 저장된 내용을 가로질러 둘러보기 위한 범위입니다. 활성 프로젝트의 기본값을 대체하지 않습니다.
- 프로젝트 안에 있지 않을 때 새 녹음과 가져오기가 들어가는 기본 위치는 `Inbox`입니다.
- 활성 live recording 세션은 캡처가 계속되는 동안 이미 여기에서 `Draft` 항목으로 나타날 수 있습니다. 녹음을 멈추면 같은 항목이 완성된 저장 항목이 됩니다.
- 프로젝트 아이콘은 system icons, 추천 emoji, custom emoji를 지원합니다.
- 저장된 녹음과 가져온 파일도 자체 이름 변경 흐름에서 제목과 아이콘을 바꿀 수 있지만, 이는 `Project Settings`와 별개입니다.
- 저장되지 않은 변경이 있는 프로젝트 설정 drawer에서 다른 프로젝트로 이동하면 Sona가 먼저 편집을 버릴지 묻습니다.

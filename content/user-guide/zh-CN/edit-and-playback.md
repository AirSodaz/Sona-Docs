当 Sona 已经生成转录分段后，右侧编辑器就是主要工作区。这里负责检查文本、对齐时间戳、校对说话人归属，以及在批量改写后恢复需要的版本。

## 适合这些场景

- 逐段检查措辞
- 在转录文本和原始时间点之间快速来回跳转
- 在导出前修正说话人标签
- 在润色、翻译或重新转录后比较 `版本快照`
- 在导出前完成搜索、合并、删除和轻量格式整理

## 开始前

- 当前已经通过 [实时录音](guide:live-record)、[批量转录](guide:batch-import) 或 [工作区、项目与 Inbox](guide:workspace-projects-and-inbox) 载入转录内容。

## 如何整理转录分段

1. 在编辑器中查看分段列表。
2. 点击时间戳跳转到对应播放位置。
3. 双击分段文本，或使用编辑按钮进入编辑状态。
4. 按 `Enter` 保存当前分段。
5. 按 `Shift + Enter` 在编辑时插入换行。
6. 使用合并按钮将当前分段与下一段合并。
7. 使用删除按钮在确认后删除分段。

## 如何校对说话人标签

1. 如果分段带有说话人徽标，点击徽标即可把整个说话人分组绑定到某个 `说话人档案`。
2. 同一个菜单里也可以展开更多全局档案，或把这个分组恢复为匿名标签。
3. 导出前如果要集中检查说话人，请从转录标题栏打开 `说话人校对`。
4. 按 `待审`、`候选建议`、`匿名`、`已识别`、`已审阅` 或 `全部` 筛选。
5. 预览每个分组的前几个片段，跳到首段，确认当前标签，应用候选档案，绑定其他档案，或恢复匿名标签。

## 如何使用版本快照

1. 当当前转录是已保存且非草稿的工作区条目时，从转录标题栏打开 `版本快照`。
2. 选择一次 `LLM 润色`、`翻译` 或 `重新转录` 等批量改写前保存的快照。
3. 比较该快照与当前转录的差异。
4. 如果只需要回退部分内容，恢复选中的差异行。
5. 如果整次改写都不需要，整份回退到该快照。

## 如何配合搜索和回放

1. 按 `Ctrl + F` 搜索当前转录内容。
2. 如果当前条目带有音频，使用播放器进行播放、暂停、跳转、倍速和音量控制。
3. 在搜索结果之间切换时，不需要离开编辑器。

## 编辑区保持哪些能力

- 转录内容始终可以按分段修改。
- 说话人修正会应用到整个说话人分组，不只是当前可见分段。
- 时间轴和播放器位置会保持联动。
- 从 `版本快照` 恢复前会先保存当前转录，所以回退本身也可以继续追溯。
- 只有在编辑某一段时，编辑工具栏才会出现。

## 补充说明

- 工具栏支持 `撤销`、`重做`、加粗、斜体、下划线和换行。
- `说话人档案` 在 [词汇与高级设置](guide:vocabulary-and-advanced-settings) 中创建。档案会帮助候选建议和自动匹配，但仍然可以人工确认。
- `版本快照` 只会显示在已保存、非草稿、且不是临时 `current` 的工作区条目上。
- 如果下一步要补充可选的 LLM 处理，可以继续看 [AI 润色与翻译](guide:ai-polish-and-translate)。
- 如果你还要重新整理或切换已保存条目，可以继续看 [工作区、项目与 Inbox](guide:workspace-projects-and-inbox)。
- 如果主要目标已经变成导出文件，直接跳到 [导出与设置](guide:export-and-settings) 即可。

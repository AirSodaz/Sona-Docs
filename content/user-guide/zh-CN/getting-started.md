如果你是第一次打开 Sona，这一页就是从安装到“本地离线转录可用”的最短路径。

## 适合这些情况

- 第一次安装桌面端应用
- 想直接使用推荐的离线初始化方案，而不是自己手动配模型
- 希望尽快进入 `实时录音` 开始第一轮可用转录

## 开始前

- Sona 可以在本机正常打开。
- 如果你准备在引导中直接下载推荐模型包，请保持网络可用。

## 先完成安装

1. 先打开 [Sona 下载页](/zh/downloads)，按你的桌面系统选择构建。如果站点暂时无法读取 release 详情，再使用 [GitHub Releases](https://github.com/AirSodaz/sona/releases/latest) 作为备用入口。
2. 启动应用。
3. 如果你是从源码构建，请改看 [项目 README](readme)。

如果遇到 Windows SmartScreen、macOS Gatekeeper 或 Linux 可执行权限提示，请先确认安装包来自 Sona 下载页或 GitHub Releases，再按系统提示处理；不要全局关闭系统安全防护。

## 完成 `首次运行设置`

1. 启动 Sona 后，等待 `首次运行设置` 出现。
2. 阅读欢迎步骤。推荐的首条成功路径是 `麦克风 -> 实时录音`。
3. 点击 `继续`。
4. 在模型步骤点击 `下载推荐模型`。如果所需模型已经存在，界面会直接显示 `继续`。
5. 等待推荐离线模型下载并解压完成。
6. 进入麦克风步骤并允许麦克风访问权限。
7. 如果权限被拒绝，请在处理系统权限提示后点击 `重新请求权限`。
8. 选择希望 `实时录音` 默认使用的麦克风。
9. 点击 `前往实时录音`。

## 完成后会发生什么

- Sona 会应用推荐的本地离线转录配置。
- 应用会切换到 `实时录音`。
- 如果设置仍未完成，提醒横幅之后还能再次打开引导。

## 完成设置后建议继续看什么

- 如果你准备马上边录边转，接着看 [实时录音](guide:live-record)。
- 如果你已经有现成文件，改看 [批量转录](guide:batch-import)。
- 如果设置反复弹出或模型仍提示缺失，直接看 [常见问题与排障](guide:faq)。

## 补充说明

- 你可以在引导过程中点击 `稍后`，之后再通过提醒横幅返回。
- 如果你主动关闭提醒横幅，它会在首页停止显示，直到设置完成。
- 后续可以在 `设置 > 模型设置` 中改用其他模型。
- 默认麦克风可以在 `设置 > 输入设备` 中重新调整。
- 如果模型下载卡住，先确认网络可用且模型托管地址可访问，再重新尝试引导；如果仍失败，到 `设置 > 模型设置` 下载或选择模型，再用 `设置 > 常规 > 诊断` 查看详情。
- 如果你之后要改用命令行批量转录，可以继续看站内的 [CLI 指南](guide:cli-guide)。

# ワークスペース、プロジェクト、インボックス

メインのエディタ表示から離れることなく、保存済みの録音ファイルやインポートファイルを整理したい場合は、「Workspace（ワークスペース）」を使用します。

## 推奨対象

- 保存したアイテムを別画面に移動することなく、一箇所から開き直す
- 「All Items（すべてのアイテム）」、「Inbox（インボックス）」、および各種プロジェクトの範囲（スコープ）で作業を分類・整理する
- 新しい録音やインポートを開始する前に、プロジェクト固有のデフォルト設定を適用する

## 最初に3つのスコープを理解する

- 「All Items（すべてのアイテム）」は、インボックスや各プロジェクトに保存されているすべてのコンテンツを横断して表示します。
- 「Inbox（インボックス）」は、まだプロジェクトに割り当てられていない録音やインポートデータを保持します。
- 各プロジェクトは、それぞれ固有の作業領域（ワークスペース）、プロジェクト別のデフォルト設定、および新しいタスクを素早く開始するためのエントリポイントを提供します。

## プロジェクトの作成と切り替え

1. 「Workspace」タブを開きます。
2. 左側のレールを使用して、「All Items（すべてのアイテム）」、「Inbox（インボックス）」、および作成済みの各プロジェクトを切り替えます。
3. クライアント、講座、定例会議など、繰り返し発生する作業に専用のスペースを割り当てたい場合は、「New Project（新規プロジェクト）」をクリックします。
4. プロジェクト名と任意の説明を入力し、作成します。新規プロジェクト作成時にはカスタムアイコンを設定できません。
5. 新しい録音やインポートファイルをそのプロジェクトへ自動的に保存したい場合は、あらかじめ「Live Record（ライブ録音）」や「Batch Import（バッチインポート）」を開始する前にプロジェクトを開いておきます。

## 保存アイテムの閲覧と整理

1. 保存済みのアイテムをクリックすると、組み込みの詳細ペインで開き、「Workspace」を表示したまま編集を継続できます。
2. 検索、フィルター、並べ替え、および表示形式の切り替え（「List View（リスト表示）」/「Grid View（グリッド表示）」/「Table View（テーブル表示）」）を使用して、表示範囲を絞り込みます。
3. 選択モードをオンにすると、アイテムをインボックスと各プロジェクト間で移動させたり、複数のアイテムを一括で削除したりできます。
4. プロジェクトのコンテキストを維持したまま「Live Record」や「Batch Import」を開始したい場合は、プロジェクトヘッダーにあるボタンを使用します。

## プロジェクト設定（Project Settings）の調整

1. プロジェクトを開き、「Project Settings」をクリックします。
2. プロジェクト名と説明を更新します。
3. 必要に応じてプロジェクトのアイコンを選択します。プロジェクトのアイコンは「新規プロジェクト」ダイアログではなく、この設定画面で編集します。
4. そのプロジェクト内で作業する際に適用するデフォルト値を選択します。「Default Summary Template（デフォルト要約テンプレート）」、「Default Translation Language（デフォルト翻訳言語）」、「Default Polish Scenario（デフォルト校正シナリオ）」、任意の「Default Polish Context（デフォルト校正文脈）」、および「Export Filename Prefix（エクスポートファイル名のプレフィックス）」。
5. プロジェクト固有の認識やクリーンアップ動作を行いたい場合は、そのプロジェクト用の「Text Replacement（テキスト置換）」や「Hotword（ホットワード）」セットのオン・オフを切り替えます。
6. 「Save（保存）」をクリックして変更を保持します。プロジェクト自体を削除し、その中のアイテムをインボックスに戻したい場合は、「Delete Project（プロジェクト削除）」をクリックします。

## お役立ちメモ

- 「All Items」は、保存済みのコンテンツを横断して閲覧するためのものであり、アクティブなプロジェクトのデフォルト設定を上書きするものではありません。
- どのプロジェクトにも入っていない状態で新しい録音やインポートを開始した場合、新しいアイテムはデフォルトの保存先である「Inbox」に保存されます。
- ライブ録音のキャプチャ実行中であっても、現在進行中のセッションは「下書き（Draft）」アイテムとしてここに表示されます。録音を停止すると、そのアイテムが完成版のデータになります。
- プロジェクトアイコンは、システムアイコン、推奨絵文字、およびカスタム絵文字をサポートしています。
- 保存された録音やインポートアイテムについても、名前変更フローを通じてタイトルやアイコンを変更できますが、これは「Project Settings」とは別の設定です。
- プロジェクトの設定ドロワーで未保存の変更がある状態で、別のプロジェクトやビューに切り替えようとすると、変更を破棄してよいか確認のメッセージが表示されます。

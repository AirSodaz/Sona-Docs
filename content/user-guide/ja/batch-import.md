# バッチインポート

すでに音声または動画ファイルがあり、Sonaにバックグラウンドで処理させたい場合は、「Batch Import（バッチインポート）」を使用します。

## 推奨対象

- 録画・録音済みの会議、講義、ポッドキャスト、またはインタビュー
- ローカルのメディアファイルからの字幕作成
- 複数のファイルをキューに追加し、順次処理を完了させていくワークフロー

## キューにファイルを追加する前に

- 「Settings > Model Settings」で「Batch Import Model（バッチインポートモデル）」を設定してください。
- ファイル形式がサポート対象のオーディオまたはビデオ形式であることを確認してください。

## ファイルをキューに追加する

1. 「Batch Import」タブを開きます。
2. ファイルをインポート領域にドラッグ＆ドロップするか、「Select File（ファイルを選択）」をクリックします。
3. 1つ以上のファイルをキューに追加します。
4. キューのサイドバーとアクティブなアイテムのステータス表示を確認します。
5. キューをさらに追加したい場合は、「Add More Files（ファイルをさらに追加）」を使用します。

## インポート動作の調整

1. 新規処理時の「Subtitle Mode（字幕モード）」や「Language（言語）」を変更したい場合は、「Parameter Settings（パラメータ設定）」をクリックします。
2. キューのステータス（`Pending（保留中）`、`Processing（処理中）`、`Complete（完了）`、または `Failed（失敗）`）を確認します。

## 処理完了後の動作

- 完了したアイテムはメインの文字起こしエディタに読み込まれます。
- そこから続けて、[編集と再生](guide:edit-and-playback)、[AI校正と翻訳](guide:ai-polish-and-translate)、[ワークスペース、プロジェクト、インボックス](guide:workspace-projects-and-inbox)、または[エクスポートと設定](guide:export-and-settings)へと進むことができます。

## お役立ちヒント

- オフラインバッチモデルが設定されていない場合、Sonaはインポートを開始する代わりにオンボーディング画面を再表示します。
- 「Settings > Model Settings」には、バッチ処理の挙動に影響を与える「VAD Buffer Size」や「Max Concurrent Transcriptions（最大同時文字起こし数）」などの設定項目があります。
- 既存のメディアから字幕ファイルを作成したい場合、「Batch Import」とエクスポート機能の組み合わせがメインルートになります。

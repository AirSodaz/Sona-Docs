# スタートガイド

Sonaを初めて起動する場合、これがローカル文字起こし環境をすばやく構築するための最短ルートです。

## 推奨対象

- 初めてデスクトップアプリをインストールする方
- 手動でのモデル選択ではなく、推奨のオフライン設定を使用したい方
- できるだけ早く「Live Record（ライブ録音）」を開始したい方

## 開始する前に

- Sonaがマシン上で正常に起動できる状態であること。
- オンボーディング中にアプリが推奨モデルパックをダウンロードできるように、インターネット接続を有効にしておくこと。

## デスクトップアプリのインストール

1. [Sona Downloads](/downloads)にアクセスし、お使いのデスクトップシステム用のビルドを選択します。ダウンロードサイトでリリース情報が読み込めない場合は、フォールバックとして[GitHub Releases](https://github.com/AirSodaz/sona/releases/latest)を使用してください。
2. アプリを起動します。
3. ソースコードからビルドする場合は、代わりに[プロジェクトのREADME](readme)を参照してください。

Windows SmartScreen、macOS Gatekeeper、またはLinuxの実行権限プロンプトが表示された場合は、インストーラーがSona DownloadsまたはGitHub Releasesから取得したものであることを確認し、OSのプロンプトに従って実行を許可してください。システム保護設定をグローバルに無効化しないでください。

## 「初回セットアップ（First Run Setup）」の完了

1. Sonaの起動時に「First Run Setup」画面が表示されるまで待ちます。
2. ウェルカムステップを確認します。Sonaが推奨する最初の実行パスは「Microphone（マイク） -> Live Record（ライブ録音）」です。
3. 「Continue（続行）」をクリックします。
4. モデルのステップで、「Download Recommended Models（推奨モデルをダウンロード）」をクリックします。必要なモデルがすでに存在する場合、代わりに「Continue」が表示されることがあります。
5. 推奨オフラインモデルのダウンロードと展開が完了するまで待ちます。
6. マイクのステップに進み、マイクへのアクセス権限を許可します。
7. マイク権限が拒否された場合は、OSの権限プロンプトから修正した後に「Try Permission Again（権限の再試行）」を使用してください。
8. Sonaの「Live Record」で使用するデフォルトマイクを選択します。
9. 「Start with Live Record（ライブ録音を開始する）」をクリックします。

## セットアップ後の動作

- Sonaはローカル文字起こし用に推奨のオフライン設定を適用します。
- アプリが「Live Record」画面で開きます。
- セットアップが完了していない場合、後からリマインダーバナーを使用してオンボーディングを再開できます。

## セットアップ完了後の次のステップ

- すぐに音声を録音したい場合は、[Live Record](guide:live-record)に進みます。
- すでに文字起こししたいファイルがある場合は、[Batch Import](guide:batch-import)に進みます。
- セットアップが何度も表示される場合や、モデルが見つからない場合は、[FAQ and Troubleshooting](guide:faq)を参照してください。

## お役立ちメモ

- オンボーディング中に「Later（後で）」をクリックし、後からリマインダーバナーで戻ることも可能です。
- リマインダーバナーを非表示にすると、セットアップが完了するまでホーム画面に表示されなくなります。
- モデルは後から「Settings > Model Settings」で変更できます。
- デフォルトのマイクは後から「Settings > Input Device」で変更できます。
- モデルのダウンロードが途中で停止した場合は、インターネット接続とモデルホストへの通信が可能か確認し、オンボーディングを再試行してください。それでも失敗する場合は、「Settings > Model Settings」でモデルをダウンロードまたは選択し、詳細は「Settings > General > Diagnostics」を開いて確認してください。
- 将来的にコマンドラインからの一括文字起こしを行いたい場合は、アプリ内の[CLI Guide](guide:cli-guide)に進んでください。

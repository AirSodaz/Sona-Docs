# 編集と再生

Sonaが文字起こしのセグメントを作成した後、エディタは文字起こし結果の確認、修正、ナビゲーション、話者の割り当ての確認、および大規模な書き換えからの復元を行うためのメインの場所になります。

## 推奨対象

- セグメントごとに言葉遣いや表現を確認・修正する
- 文字起こしテキストと元の音声の再生位置（タイムスタンプ）を相互にジャンプする
- 文字起こしデータを外部へ書き出す前に、話者ラベルを修正する
- LLM校正、翻訳、または再文字起こしなどの変更後に、「Version Snapshots（バージョン履歴）」を比較する
- エクスポートする前に、文字起こしコンテンツの検索、結合、削除、または簡易的な書式設定を行う

## 編集を始める前に

- [ライブ録音](guide:live-record)、[バッチインポート](guide:batch-import)、または[ワークスペース、プロジェクト、インボックス](guide:workspace-projects-and-inbox)から文字起こしコンテンツを読み込んでください。

## 文字起こしセグメントの確認と変更

1. エディタ内のセグメント一覧を確認します。
2. タイムスタンプをクリックすると、その時点の音声再生位置にシークします。
3. セグメントのテキストをダブルクリックするか、編集アクションを使用して編集モードに入ります。
4. `Enter` キーを押して、現在のセグメントを保存します。
5. `Shift + Enter` キーを押して、現在のセグメントをカーソル位置で分割します。
6. 結合アクションを使用して、現在のセグメントと次のセグメントをまとめます。
7. 削除アクションを使用して、確認後にセグメントを削除します。

## 話者ラベルの確認

1. セグメントに話者バッジがある場合、バッジをクリックして話者グループ全体を「Speaker Profile（話者プロファイル）」に割り当てることができます。
2. 同じメニューを使用して、グローバルに追加された他のプロファイルを表示させたり、そのグループを匿名のラベルに戻したりできます。
3. エクスポート前に集中的に話者を確認したい場合は、文字起こし画面の上部にある「Speaker Review（話者確認）」を開きます。
4. 「Needs review（確認が必要）」、「Suggestions（提案）」、「Anonymous（匿名）」、「Identified（識別済み）」、「Reviewed（確認済み）」、または「All（すべて）」でフィルタリングします。
5. 各グループの最初のセグメントをプレビューし、一致する最初のセグメントにジャンプしたり、現在のラベルを確定したり、提案された候補を適用したり、別のプロファイルを割り当てたり、グループを匿名にリセットしたりします。

## バージョン履歴（スナップショット）の利用

1. 文字起こしがワークスペースに保存された（下書き以外）アイテムである場合、文字起こし画面の上部にある「Version Snapshots」を開きます。
2. 「LLM Polish（LLM校正）」、「Translate（翻訳）」、または「Re-transcribe（再文字起こし）」などの一括書き換え操作を行う前に保存されたスナップショットを選択します。
3. スナップショットを現在の文字起こしと比較します。
4. 書き換えの一部のみをロールバックしたい場合は、変更された特定の行を選択して復元（Restore）します。
5. 書き換え全体を破棄したい場合は、文字起こし全体を元の状態に戻します。

## 検索と再生

1. `Ctrl + F` を押して、文字起こし内を検索します。
2. オーディオファイルが利用可能な場合は、オーディオプレーヤーを使用して再生、一時停止、シーク、再生速度の変更、音量の調整を行います。
3. エディタから移動することなく、一致するセグメント間をジャンプします。

## エディタで常に適用される基本動作

- 文字起こしテキストはセグメント単位で編集可能です。
- 話者の修正は、表示されている単一のセグメントだけでなく、話者グループ全体に一括して適用されます。
- 再生位置と文字起こしテキストの追従は、タイムスタンプによって同期されます。
- 「Version Snapshots」から復元する際、現在の文字起こしが自動的に保存されるため、ロールバック操作自体も取り消しが可能です。
- エディタのツールバーは、セグメントの編集時のみ表示されます。

## お役立ちヒント

- ツールバーは、「Undo（元に戻す）」、「Redo（やり直し）」、「Bold（太字）」、「Italic（斜体）」、「Underline（下線）」、およびセグメント分割をサポートしています。
- 「Speaker Profiles」は、[語彙と詳細設定](guide:vocabulary-and-advanced-settings)で作成できます。プロファイルは候補の提案や自動マッチングに役立ちますが、手動での確認や修正も可能です。
- 「Version Snapshots」は、下書きや一時的な「現在のセグメント（current）」の文字起こしではない、保存済みのワークスペースアイテムに対してのみ表示されます。
- 編集後にLLMによる任意の校正や翻訳を追加したい場合は、[AI校正と翻訳](guide:ai-polish-and-translate)に進んでください。
- 保存されたアイテムをエディタ周辺で開き直したり再整理したりしたい場合は、[ワークスペース、プロジェクト、インボックス](guide:workspace-projects-and-inbox)に進んでください。
- 作業が完了し、主にファイルをSonaからエクスポートしたい場合は、[エクスポートと設定](guide:export-and-settings)へ進んでください。

# ライブ録音

音声を録音しながら、文字起こしのセグメントをリアルタイムで表示させたい場合は、「Live Record（ライブ録音）」を使用します。

## 推奨対象

- 会議、インタビュー、講義、またはボイスメモ
- 録音中もタイムスタンプを紐付けたままにしておきたいワークフロー
- 録音停止後、表示されているライブ下書きを含めて、すべてをSonaのメインワークスペースに保存しておきたい方

## 開始する前に

- [スタートガイド](guide:getting-started)を完了するか、あるいは「Settings > Model Settings」で「Live Record Model（ライブ録音モデル）」を手動で設定してください。
- マイクから録音する場合は、オペレーティングシステムでマイクのアクセス権限が許可されていることを確認してください。

## ライブセッションの録音

1. 「Live Record」タブを開きます。
2. 録音を開始する前に、ドロップダウンから入力ソース（「Microphone（マイク）」または「Desktop Audio（デスクトップオーディオ）」）を選択します。
3. 「Start Recording（録音開始）」をクリックします。
4. Sonaがオーディオをキャプチャしている間、波形とタイマーを確認します。
5. セッションを終了せずに一時停止したい場合は、「Pause（一時停止）」を使用します。
6. 録音を完了して保存したい場合は、「Stop（停止）」を使用します。

## 録音中の動作調整

1. 「Subtitle Mode（字幕モード）」や「Language（言語）」を変更したい場合は、「Parameter Settings（パラメータ設定）」をクリックします。
2. リアルタイムでの使用中にフローティング字幕ウィンドウを表示させたい場合は、「Live Caption（ライブ字幕）」をオンにします。
3. 最前面表示、クリック透過、サイズ、幅、色、または起動時の動作など、字幕の挙動を変更したい場合は、「Settings > Subtitle Settings」を開きます。
4. 録音フローではなく、フル機能の「Live Caption」機能をお探しの場合は、[ライブ字幕と音声入力](guide:live-caption-and-voice-typing)に進んでください。

## 「ライブ字幕（Live Caption）」の用途

- 「Live Caption」は、「Live Record」ページにある「System Audio Captions（システムオーディオ字幕）」トグルのことであり、主にシステムオーディオのフローティング字幕ウィンドウが必要な場合に便利です。
- 録音を開始することなく単独でオンにできます。その後「Live Record」を開始した場合、両方を並行して実行できます。
- 「Settings > Subtitle Settings」で、起動時の動作、最前面表示、クリック透過、フォントサイズ、幅、色、および背景の透過度を制御します。
- ページ内の切り替えスイッチと設定レイヤーの違いについて詳しく知りたい場合や、「Voice Typing（音声入力）」も必要な場合は、直接[ライブ字幕と音声入力](guide:live-caption-and-voice-typing)に進んでください。
- 「Live Caption」をオンにしてもウィンドウが表示されない場合は、[FAQとトラブルシューティング](guide:faq)を確認してください。

## 録音中および録音後に得られるもの

- 文字起こしのセグメントが右側のエディタに表示されます。
- キャプチャの実行中、アクティブなセグメントはライブ録音の状態に追従します。
- キャプチャの実行中であっても、Sonaはすでにそのセッションを「下書き（Draft）」アイテムとして[ワークスペース、プロジェクト、インボックス](guide:workspace-projects-and-inbox)に表示できます。
- 録音が停止すると、文字起こしテキストは編集、校正、翻訳、エクスポートの対象となり、後から[ワークスペース、プロジェクト、インボックス](guide:workspace-projects-and-inbox)で再度開くことができます。Sonaは別の新規保存ファイルを作成するのではなく、この下書きアイテムを完成版として上書き保存します。
- 事前にプロジェクト内で作業を開始していなかった場合、保存されたアイテムは通常「Inbox（インボックス）」に保存されます。

## お役立ちヒント

- デフォルトでは `Ctrl + Space` でライブ録音の開始と停止が行えます。
- 録音実行中は `Space` キーで一時停止と再開を切り替えられます。
- 「Parameter Settings」は、「Subtitle Mode」や「Language」などの文字起こし時の挙動のみを設定するものであり、LLMによる校正（LLM Polish）のワークフロー全体を設定するものではありません。
- モデルが見つからないというエラーが表示される場合は、オンボーディングを再実行するか、「Settings > Model Settings」でモデルを設定してください。
- キャプチャ完了後の次のステップとして適しているのは、通常[編集と再生](guide:edit-and-playback)です。

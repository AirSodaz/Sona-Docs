當 Sona 已經生成轉錄分段後，右側編輯器就是主要工作區。這裡負責檢查文字、對齊時間戳記、校對說話者歸屬，以及在批次改寫後復原需要的版本。

## 適合這些場景

- 逐段檢查措辭
- 在轉錄文字和原始時間點之間快速來回跳轉
- 在匯出前修正說話者標籤
- 在潤飾、翻譯或重新轉錄後比較 `版本快照`
- 在匯出前完成搜尋、合併、刪除和輕量格式整理

## 開始前

- 目前已經透過 [即時錄音](guide:live-record)、[批次轉錄](guide:batch-import) 或 [工作區、專案與 Inbox](guide:workspace-projects-and-inbox) 載入轉錄內容。

## 如何整理轉錄分段

1. 在編輯器中查看分段列表。
2. 點擊時間戳記跳轉到對應播放位置。
3. 雙擊分段文字，或使用編輯按鈕進入編輯狀態。
4. 按 `Enter` 儲存目前分段。
5. 按 `Shift + Enter` 在游標位置拆分目前分段。
6. 使用合併按鈕將目前分段與下一段合併。
7. 使用刪除按鈕在確認後刪除分段。

## 如何校對說話者標籤

1. 如果分段帶有說話者徽章，點擊徽章即可把整個說話者分組繫結到某個 `說話者檔案`。
2. 同一個選單裡也可以展開更多全域檔案，或把這個分組復原為匿名標籤。
3. 匯出前如果要集中檢查說話者，請從轉錄標題列打開 `說話者校對`。
4. 按 `待審`、`候選建議`、`匿名`、`已辨識`、`已審閱` 或 `全部` 篩選。
5. 預覽每個分組的前幾個片段，跳到首段，確認目前標籤，應用候選檔案，繫結其他檔案，或復原匿名標籤。

## 如何使用版本快照

1. 當目前轉錄是已儲存且非草稿的工作區條目時，從轉錄標題列打開 `版本快照`。
2. 選擇一次 `LLM 潤飾`、`翻譯` 或 `重新轉錄` 等批次改寫前儲存的快照。
3. 比較該快照與目前轉錄的差異。
4. 如果只需要回退部分內容，復原選中的差異行。
5. 如果整次改寫都不需要，整份回退到該快照。

## 如何配合搜尋和回放

1. 按 `Ctrl + F` 搜尋目前轉錄內容。
2. 如果目前條目帶有音訊，使用播放器進行播放、暫停、跳轉、倍速和音量控制。
3. 在搜尋結果之間切換時，不需要離開編輯器。

## 編輯區保持哪些能力

- 轉錄內容始終可以按分段修改。
- 說話者修正會應用到整個說話者分組，不只是目前可見分段。
- 時間軸和播放器位置會保持連動。
- 從 `版本快照` 復原前會先儲存目前轉錄，所以回退本身也可以繼續追溯。
- 只有在編輯某一段時，編輯工具列才會出現。

## 補充說明

- 工具列支援 `復原`、`重做`、粗體、斜體、底線和拆分分段。
- `說話者檔案` 在 [詞彙與進階設定](guide:vocabulary-and-advanced-settings) 中建立。檔案會幫助候選建議和自動匹配，但仍然可以人工確認。
- `版本快照` 只會顯示在已儲存、非草稿、且不是臨時 `current` 的工作區條目上。
- 如果下一步要補充可選的 LLM 處理，可以繼續看 [AI 潤飾與翻譯](guide:ai-polish-and-translate)。
- 如果您還要重新整理或切換已儲存條目，可以繼續看 [工作區、專案與 Inbox](guide:workspace-projects-and-inbox)。
- 如果主要目標已經變成匯出檔案，直接跳到 [匯出與設定](guide:export-and-settings) 即可。

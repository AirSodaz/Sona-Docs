如果您是第一次打開 Sona，這一頁就是從安裝到「本機離線轉錄可用」的最短路徑。

## 適合這些情況

- 第一次安裝桌面端應用程式
- 想直接使用推薦的離線初始化方案，而不是自己手動配模型
- 希望儘快進入 `即時錄音` 開始第一輪可用轉錄

## 開始前

- Sona 可以在本機正常打開。
- 如果您準備在引導中直接下載推薦模型包，請保持網路可用。

## 先完成安裝

1. 先打開 [Sona 下載頁](/downloads)，按您的桌面系統選擇建構。如果站台暫時無法讀取 release 詳情，再使用 [GitHub Releases](https://github.com/AirSodaz/sona/releases/latest) 作為備用入口。
2. 啟動應用程式。
3. 如果您是從原始碼建構，請改看 [專案 README](readme)。

如果遇到 Windows SmartScreen、macOS Gatekeeper 或 Linux 可執行權限提示，請先確認安裝包來自 Sona 下載頁或 GitHub Releases，再按系統提示處理；不要全域關閉系統安全防護。

## 完成 `首次執行設定`

1. 啟動 Sona 後，等待 `首次執行設定` 出現。
2. 閱讀歡迎步驟。推薦的首條成功路徑是 `麥克風 -> 即時錄音`。
3. 點擊 `繼續`。
4. 在模型步驟點擊 `下載推薦模型`。如果所需模型已經存在，介面會直接顯示 `繼續`。
5. 等待推薦離線模型下載並解壓縮完成。
6. 進入麥克風步驟並允許麥克風存取權限。
7. 如果權限被拒絕，請在處理系統權限提示後點擊 `重新請求權限`。
8. 選擇希望 `即時錄音` 預設使用的麥克風。
9. 點擊 `前往即時錄音`。

## 完成後會發生什麼

- Sona 會應用推薦的本機離線轉錄設定。
- 應用程式會切換到 `即時錄音`。
- 如果設定仍未完成，提醒橫幅之後還能再次打開引導。

## 完成設定後建議繼續看什麼

- 如果您準備馬上邊錄邊轉，接著看 [即時錄音](guide:live-record)。
- 如果您已經有現成檔案，改看 [批次轉錄](guide:batch-import)。
- 如果設定反覆彈出或模型仍提示缺少，直接看 [常見問題與疑難排解](guide:faq)。

## 補充說明

- 您可以在引導過程中點擊 `稍後`，之後再透過提醒橫幅返回。
- 如果您主動關閉提醒橫幅，它會在首頁停止顯示，直到設定完成。
- 後續可以在 `設定 > 模型設定` 中改用其他模型。
- 預設麥克風可以在 `設定 > 輸入裝置` 中重新調整。
- 如果模型下載卡住，先確認網路可用且模型代管地址可存取，再重新嘗試引導；如果仍失敗，到 `設定 > 模型設定` 下載或選擇模型，再用 `設定 > 常規 > 診斷` 查看詳情。
- 如果之後需要獨立 CLI 的檔案/即時轉錄、模型管理、歷史/匯出/備份自動化，或本機 API 服務，可以繼續看站內的 [CLI 指南](guide:cli-guide)。
